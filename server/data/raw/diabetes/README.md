# Diabetes Dataset

## Dataset Information
- **Source**: [Mendeley Data - Diabetes Dataset](https://data.mendeley.com/datasets/wj9rwkp9c2/1)
- **Format**: CSV

## Attribute Information

| Column Name | Description | Data Type |
|-------------|-------------|-----------|
| ID | Unique patient identifier | Numeric |
| No_Pation | Number of patient | Numeric |
| Gender | Patient gender (Gender) | Categorical |
| AGE | Age of the patient in years (Age) | Numeric |
| Urea | Blood urea level (Urea) | Numeric |
| Cr | Creatinine ratio (Creatinine) | Numeric |
| HbA1c | Glycated hemoglobin level (HbA1c) | Numeric |
| Chol | Total cholesterol level (Cholesterol) | Numeric |
| TG | Triglycerides level (Triglycerides) | Numeric |
| HDL | High-density lipoprotein cholesterol level (HDL) | Numeric |
| LDL | Low-density lipoprotein cholesterol level (LDL) | Numeric |
| VLDL | Very low-density lipoprotein cholesterol level (VLDL) | Numeric |
| BMI | Body Mass Index (BMI) | Numeric |
| CLASS | Diabetes status (N = Non-Diabetic, P = Predict-Diabetic, Y = Diabetic) | Categorical |
