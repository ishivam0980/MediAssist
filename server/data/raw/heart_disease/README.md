# Heart Disease Dataset

## Dataset Information

- **Source**: [Kaggle - Heart Disease Dataset](https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset)
- **Format**: CSV

## Attribute Information

| Column Name | Description                                                       | Data Type   |
| ----------- | ----------------------------------------------------------------- | ----------- |
| age         | Age in years (Age)                                                | Numeric     |
| sex         | Gender (1 = male, 0 = female) (Sex)                               | Categorical |
| cp          | Chest pain type (0-3) (Chest Pain Type)                           | Categorical |
| trestbps    | Resting blood pressure (mm Hg) (Resting BP)                       | Numeric     |
| chol        | Serum cholesterol (mg/dl) (Cholesterol)                           | Numeric     |
| fbs         | Fasting blood sugar > 120 mg/dl (1 = true; 0 = false) (Fasting Blood Sugar) | Categorical |
| restecg     | Resting electrocardiographic results (0-2) (Resting ECG)          | Categorical |
| thalach     | Maximum heart rate achieved (Max HR)                              | Numeric     |
| exang       | Exercise induced angina (1 = yes; 0 = no) (Ex. Angina)            | Categorical |
| oldpeak     | ST depression induced by exercise relative to rest (Oldpeak)      | Numeric     |
| slope       | The slope of the peak exercise ST segment (0-2) (Slope)           | Categorical |
| ca          | Number of major vessels colored by fluoroscopy (0-4) (Vessels)    | Categorical |
| thal        | Thalassemia (0 = normal; 1 = fixed defect; 2 = reversible defect) (Thal) | Categorical |
| target      | Heart disease (1 = present; 0 = absent)                           | Categorical |
